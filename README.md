Este projeto foi desenvolvido para fins educacionais utilizando o framework Vue.js e a biblioteca de estilização TailwindCSS. O objetivo do trabalho foi criar um sistema simples de listagem de filmes e avaliação, onde é possível registrar filmes e atribuir notas decimais entre 0 e 10.

O sistema permite organizar e visualizar os filmes cadastrados juntamente com suas avaliações, servindo como prática de desenvolvimento web, lógica de programação e uso de frameworks modernos de desenvolvimento.

Este trabalho foi desenvolvido como projeto final da disciplina Algoritmos e Lógica de Programação.

Este trabalho contou com a contribuição de quatro alunos: José Otavio, Jadson Carvalho, Ronalty Hauan e Caua Santos.

O projeto foi desenvolvido sob supervisão do professor Anthony Irlan Marques Luz.

Aviso de uso de IA: ferramentas de inteligência artificial foram utilizadas como apoio para esclarecimento de dúvidas, auxílio na orientação durante o desenvolvimento do projeto. O uso dessas ferramentas teve apenas caráter de suporte educacional.
